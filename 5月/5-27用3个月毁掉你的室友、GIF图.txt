	如何用3个月毁掉你的室友？（大学生版）

	链接：http://zhuanlan.zhihu.com/p/21249109

	如果你的室友是一位成绩优异的三好学生，而你又恰巧是一个毫无底线的心机婊，本文会告诉你，如何利用3个月的时间，毁掉室友的人生。





	很简单，在准备阶段，你需要先让室友成为你最铁的哥们，最亲切的闺蜜，可以在你面前卸下伪装嚎啕大哭的那种。


	偶尔向他透露一些自己的小秘密，并明确表示这个秘密你只告诉了他一个人。吃饭带上他，自习带上他，逛街带上他。就好像你们是失散多年的亲兄妹。

没人会受得住如此蜜糖般的诱惑，时间一长，室友一定会对你肝胆相照。
好了，前期准备工作结束，下面就要开始你宏大的计划了。





	首先，你必须要让他对“勤奋学习”这件事情的作用产生怀疑。


	你需要在适当的时候，向室友灌输这样的思想：“你看那些富二代，就算全班倒数，毕业之后照样能当上CEO，分分钟开跑车住别墅。就算再不济，走个后门也能进国企、做公务员，比我们这些穷潘扛叩讲恢哪里去！”
	

说这话时，你一定要做出十分无奈的样子，最好再举几个例子，比如你的某个远方表亲，微积分补考了三次都没过，也不妨碍人家在某大企业混得风声水起。

你们关系那么好，室友一定不会激烈反驳你。所以接下来要赶紧趁热打铁，抛出你的结论：

读书有什么卵用？到最后还不是拼爹？辛苦四年也比不过别人爹妈一句话。既然如此，还不如好好享受难得的大学生活。

有理有据，让人信服。努力拼搏多费劲，哪有享受生活来得舒服。

但这还不够，十几年的寒窗苦读已经把室友变成了学习机器。让这样冥顽不灵的三好学生堕入黑暗，不可能只用嘴上说说这么简单。



	

所以接下来要进行第二步计划：和他一起去自习。
	从早上开始，他去图书馆，你也去图书馆；他去自习室，你也要形影不离。找个位置坐下来，装模作样认真学习一会儿，时间不用太长，半小时足够。

之后你就要开始行动了。掏出手机，找一些吸引人的搞笑视频，或者不长不短的微小说，极力推荐给室友，他的目光多停留在手机上一秒，你离胜利就近一分。瓦解他的专注力，绝不能让他在知识的海洋里肆意畅游。

这招极其高明，既能完美耽误对方的学习时间，又能增进双方的友谊，两个人看着手机嘻嘻哈哈，一举两得。

在你的努力之下，室友已经荒废了一上午。吃过午饭后，你还要记得阻止他睡午觉，让他困倦不堪，这样他就会很容易在上课时打瞌睡，再把下午完美地荒废过去。

到了晚上，你要和室友聊聊最新的综艺节目和电视剧里的情节。他整天都在忙学习，当然是不知道的。这时你一定要做出十分惊讶的表情：“什么？连这你都不知道？不会吧？你不能死读书呀，要了解一些最近发生的事情嘛，我这是为你好哦。”

室友心里一定会发毛：我是不是不太合群啊？别人的聊天内容我都听不懂，看来我也应该去补一下，这样才能和别人有共同语言啊。

如果过了一阵，你发现室友也开始说一些流行词语，对电视剧里男女主角也能品头论足一番的话，恭喜你，你已经成功了一半了。


	经过一个月的循环，室友的学习时间大幅缩短，他的成绩自然会有明显下降，这对他来说是个不小的打击。也许他已经决心下个月开始好好学习了，而这是你无论如何也不想看到的。

不过不要急，你之前所做的，只是在摧毁他的习惯而已。接下来，是时候该摧毁他的思想了。








	毁掉思想，先要从毁掉生活开始。


	一日之计在于晨。起床越晚，这一天就越容易被荒废，所以你要想尽办法让室友睡过头，绝不能让他有早起的习惯。

比如临近半夜塞给他最新一集的美剧，或者推荐一部情节跌宕起伏的电影，再或者来一次别开生面的宿舍卧谈会。只要消耗掉一两个小时，第二天他一定没办法按时起床。

买个遮光性能良好的窗帘，号召宿舍几个人一起睡懒觉。在这样的环境下，仍然能坚持早起的人实在是凤毛麟角。

生活习惯伴随着心态变化。当消灭了室友早起的习惯后，他的心态也会随之变得更加懒惰，自制力也会下降。

是时候让他变得再浮躁一些了。

金钱刺激，永远是产生浮躁、腐蚀人性的最好办法。你要让他在一次又一次的打击中，发现原来金钱能买来快乐，也能买来幸福，而自己却一无所有。

想做到这一点，你需要先向室友普及一些知识：商品不只有地摊货，连内裤都有两千块钱一条的，一把雨伞也可以值四位数，上万元的羊绒围巾真的又轻便又漂亮又暖和。


	当然，这还不够。每个班级或多或少都有一些家境殷实的同学，你要想办法和其中一个搞好关系，并在时机成熟时，和他出来吃顿饭，记得带上你的室友。

这顿饭的目的只有一个：让室友发现自己是多么LOW，而对方的生活却是多么丰富多彩。如果你的这位同学曾经环游过世界，那就让他讲讲各国的风土人情，聊聊气吞山河的异域风光，不经意间说出酒店入住一晚的昂贵价格。

谈完旅游，再谈谈未来。VC、A轮B轮、种子用户，这些术语打着滚从你同学口中喷涌而出，室友一句也听不懂。
	不过没关系，他也不需要听懂，因为他已经明白了一个道理：

人的眼界，是由他所在的层次决定的，而高层次，是用高消费换来的。没有钱，一切都没有任何卵用。

室友之前从未想过，明明还没有踏入工作岗位，人与人之间的差距竟然已经如此之大。事实似乎并不像家长和老师反复强调的，只要好好学习就可以出人头地。

他迷茫了。

迷茫时，人通常会面临两种选择：一种是化悲愤为动力，以跻身上层社会为目标，发奋图强；另一种是自暴自弃，一蹶不振，从此怨天尤人。

你可是要以毁了室友为己任的心机婊，当然要让他走后面一条路。


	所以在接下来的时间里，你要继续向室友推销你的“宿命论”：人命天注定，生下来就很难改变。那些出身好的人，是我们这些潘恐掌湟簧也无法超越的。做人嘛，最重要的还是开心，凡事差不多就得了，何必和自己过不去呢。

这些话说进了室友的心坎里。他也和你一样，不再热衷于考个高分，开始学会偷懒，过上了今朝有酒今朝醉的放浪生活。

你很快就要成功了。

	

	还有最后一件事要做：

用一些事物，去填补他空虚的内心。
	

比如适时向室友推荐几个欲罢不能的游戏，让他在虚拟世界 里重新找回自尊心，体验手握利刃斩鬼诛神的快感。

比如牵线搭桥，让室友享受爱情的甜蜜。总之，要让他在学习之外，找到强有力的精神支柱，度过充实而又无用的大学四年。

这样的状态持续到毕业，他一定会成钱钟书《围城》笔下的方鸿渐：“你不讨厌，但是全无用处。”

复仇计划至此已经接近尾声了，但你还是不解气，你想看看他真正意义上一无所有的样子。

于是你来到了第三阶段。


	爱财之心，人皆有之。
	尤其是室友这种刚被金钱打击过的人，心气浮躁，更容易误入歧途。这时你发现某公司正在招收大学生兼职，薪酬还不错，你赶忙推荐给室友。

既能赚钱，又能增加实习经历，何乐而不为。于是室友去面试了，公司要求他下周开始上班。

兼职第一天，室友就被老板一顿好骂，简单的资料整理都做不好，只会帮倒忙。

室友很郁闷，回到宿舍自然要找你吐露心声。关键的一步来了：你一定要站在室友这边，尽自己所能去鄙视那家公司。老板混账，同事混账，公司混账，一切都是他们的错，而室友没有任何责任。

不是室友不够格，而是这家公司太LOW。在你的怂恿下，室友和老板吵了一架，摔门而去。

这家公司不行，就换另外一家。室友很快找了另一份实习，但工作同样不顺心。你继续向他灌输同样的观点：一切都是别人的错。他再次炒了老板的鱿鱼。

如此往复，你最终剥夺了他的最重要的一个特质：自省。

失去自省，意味着停止反思，这也阻断了他在大学阶段最后一丝改邪归正的可能。即便若干年后他恍然大悟，期间被耽误的青春也绝非时间与金钱可以丈量。

就这样，在你潜移默化的洗脑之下，室友已经拥有了浮躁、自大、拜金、怨天尤人、眼高手低等若干缺点。

而这些缺点，每一个都是致命的。






	恭喜你，你已经成功毁掉了你的室友。

只不过，在毁掉他的同时，你也毁掉了你自己。





	

故事到这里，讲完了。
	文章中的“室友”，我更愿意把他解读为你刚刚进入大学时，尚未沉沦的样子。

那时的你还意气风发，想着改变世界，心中仍存斗志。大学生活在你眼中是纯粹的、神圣而不可侵犯的。

但四年的时光，足够把一个人雕刻得面目全非。

进了大学，你慢慢开始发现社会并没有自己想象的那样公平，有些同学生来家境优渥，自己朝思暮想、值得为之奋斗终身的生活状态，他们却唾手可得。

你又发现努力似乎并非唯一的办法，有很多充满诱惑的道路，而用功读书永远是看上去最傻最艰难的一条。

你心中愤懑，又迷茫万分。想要寻求改变，又不知该从何下手。

而大学相对轻松的生活作息，正一点一点蚕食你的意志力。
	你开始变得懒惰、得过且过，你开始学会了用借口去搪塞和安慰自己，拒绝与内心的灵魂坦诚相待。

那时的你还不知道：优秀之人，各有各的闪光点，而平庸的人，却大抵相同。

从踏入大学校门的那一刻开始，你便是孑然一身，从此没人再会强迫你，也不会有人告诉你究竟哪条路才会通向光明。

一切的决定、后果，都要你自己来承担。

我已经毕业很多年了，但仍会在某些夜晚，在睡梦中惊坐而起，想起大学时曾被虚度的时光。

我曾不止一次祈祷，让我回到刚刚踏入校园的那一刻，我愿意用20年阳寿去换。但我知道上帝对每个人都是公平的，我永远也不可能得到第二次机会。

这是我的肺腑之言，只说给你听。我不想在几年之后，你也同我一样追悔莫及。

不求成功，不求优秀，不求开天辟地造出个什么绚烂的世界。

只求无悔。




	（完）

android-gif-drawable的使用
1.获取本地gif图片
	添加依赖：compile 'pl.droidsonroids.gif:android-gif-drawable:1.1.16'
	xml
    		<pl.droidsonroids.gif.GifImageView
        		android:layout_width="match_parent"
        		android:layout_height="wrap_content"
        		android:src="@drawable/g"
        	/>
2.获取网络gif图片
	xml：<pl.droidsonroids.gif.GifImageView
	权限：<uses-permission android:name="android.permission.INTERNET"></uses-permission>
	异步任务类的jar
	加载：加载网络gif图片关健的代码是文档中提供的      
		GifDrawable drawable = new GifDrawable(byte[]);
	然后调用GifImageView中 实例类中的setBackgroundDrawable(drawable)方法 

 
	详细代码：

    private GifImageView network_gifimageview;//图片控件
    private AsyncHttpClient asyncHttpClient = new AsyncHttpClient();//获取网络数据
    private ProgressDialog dialog;//进度对话框获取网络gif图片
	{
        network_gifimageview = (GifImageView) findViewById(R.id.network_gifimageview);
        dialog = ProgressDialog.show(this, "加载中", "加载网络图片中");
        asyncHttpClient = new AsyncHttpClient();
        asyncHttpClient.get("http://cdn.duitang.com/uploads/item/201311/20/20131120213622_mJCUy.thumb.600_0.gif",
            new AsyncHttpResponseHandler() {
                @Override
                public void onSuccess(int arg0, Header[] arg1,byte[] arg2) {
                    GifDrawable drawable = null;
                    try {
                        drawable = new GifDrawable(arg2);
                    } catch (IOException e) {
                        e.printStackTrace();
                    }
                    network_gifimageview.setBackgroundDrawable(drawable);
                    dialog.dismiss();
                }

                @Override
                public void onFailure(int arg0, Header[] arg1,byte[] arg2, Throwable arg3) {
                    // TODO Auto-generated method stub
                    Toast.makeText(getApplicationContext(),
                            "加载网络图片出错", Toast.LENGTH_SHORT).show();
                    dialog.dismiss();
                }
            });
    }
